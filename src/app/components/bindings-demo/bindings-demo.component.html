One-way from data source to view target

Syntax

{{expression}}
[target] = "expression"
bind-target = "expression"

Template:

<input class="form-control" type="text" [value]="someProperty" placeholder="Property binding [value]='someProperty'"><br>
<input class="form-control" type="text" [(ngModel)]="someProperty" placeholder="Two way binding"><br>
<strong>someProperty value:</strong>{{someProperty}}

Example

One-way from view target to data source

Syntax

(target) = "statement"
on-target = "statement"

Template: 

<input class="form-control" type="text" (input)="someProperty2 = $event.target.value" placeholder="Property binding (input)='someProperty = $event.target.value'"><br>
<input class="form-control" type="text" [(ngModel)]="someProperty2" placeholder="Two way binding"><br>
<strong>someProperty value:</strong>{{someProperty2}}
