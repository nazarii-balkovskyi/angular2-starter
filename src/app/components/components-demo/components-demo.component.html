Components demo

Each Angular component requires a single @Component annotation. The @Component annotation specifies when a component is instantiated, and which properties and hostListeners it binds to.

When a component is instantiated, Angular:

  • creates a shadow DOM for the component.
  • loads the selected template into the shadow DOM.
  • creates all the injectable objects configured with providers and viewProviders.

Input and output properties

Here we have a voters example.

Code example (Vote component):

@Component({
    selector: 'vote',
    template: require('./vote.component.html')
})
export class VoteComponent {
    private _voterName: string;
    @Output() voted: EventEmitter<boolean> = new EventEmitter<boolean>();
    private voteResult: boolean;
    private done: boolean = false;

    @Input()
    set voterName(value: string) {
        this._voterName = (value && value.trim()) || '<no name set>';
    }

    vote(voted: boolean) {
        this.voted.emit(voted);
        this.voteResult = voted;
        this.done = true;
    }
}
